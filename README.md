# uniapp-music
**此项目是基于网易云音乐API服务开发的一款多端聚合云音乐，里面有账号密码登录和网易云音乐扫码登录、音乐播放、个人中心、多个平台的音乐搜索（QQ音乐、酷狗音乐、酷我音乐等）。**
